# YoRHa Chess Module

A tactical, unrestricted chess sandbox designed for analyzing board states and training against an adaptive engine. 

**[Initiate Web Module](https://velgoh.github.io/AdaptiveChessGuide/)**

## System Features
* **Adaptive Elo Tracking:** The internal engine automatically scales its difficulty (Elo) based on the precision of the opponent's moves.
* **Unrestricted Sandbox:** Freely move pieces for both White and Black without being locked into turn-based restrictions. Perfect for testing specific scenarios or studying openings.
* **Live Evaluation:** Constant monitoring of the board state with real-time evaluation scores (Eval: +0.0).
* **Tactical UI:** Full YoRHa aesthetic integration (Beige UI overlay, sharp geometrics, and monospace data readouts).

## Usage
Simply click the link above to load the web module directly in your browser. No local installation required.
